## App development practice from the book "Android Programming: The Big Nerd Ranch Guide".
### Application #2. CriminalIntent.

App with data store for events -- office criminal actions.
User can create, edit and delete an even -- criminal activity.
Events consist of few parameters, like description, date of the event, picture.
It's also possible to add crime's suspects from user's address book (and call afterwards).
Also user can export and send event report via mail.
